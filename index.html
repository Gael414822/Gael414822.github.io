1. HTML: La Estructura

¿What is HTML? Son los bloques básicos de lego. No importa el color, importa que uno es un bloque, otro es una pieza plana y otro es una ventana. HTML le dice al navegador: "Aquí va una pieza de este tamaño".

Ref: MDN Web Docs - HTML Basics

Head y Body: El Head es como la etiqueta del precio o las instrucciones de la caja: info importante que no se ve. El Body es el juguete armado que puedes tocar.

Ref: W3Schools - HTML Structure

Etiquetas de Organización

  • Header: La gorra o parte de arriba.
  • Section: Un cajón con cosas parecidas.
  • Article: Una noticia que se cuenta sola.
  • Aside: Una nota extra al margen.
  • Footer: Los zapatos de la página.

Ref: HTML5 Semantic Elements

2. CSS

¿What is CSS? Si HTML es el esqueleto, CSS es la piel, el pelo y la ropa. Sirve para que el esqueleto no asuste y se vea con colores.

Ref: CSS Design Basics

¿How to apply CSS? 1. Inline: Pegado a la etiqueta. 2. Internal: Arriba en el Head. 3. External: En un archivo .css aparte.

Ref: W3Schools - Adding CSS

Selectors, Class & ID

Un Selector es el nombre de a quién quieres cambiar. Una Rule es la orden que das.

  • Tag Selector: Afecta a todos (ej: todos los p).
  • Class Selector: Para un grupo (se usa .nombre). Es como un uniforme para muchos.
  • Id Selector: Para UNO solo (se usa #nombre). Es como un DNI o nombre propio único.
  • Tag and Class: Es más específico, como decir "solo los párrafos que tengan uniforme".

Ref: MDN - CSS Selectors Guide

Box Model & Media Queries

Box Model: Todo es una caja. Tienes el Contenido, el Padding (papel burbuja), el Borde (caja de cartón) y el Margin (espacio entre cajas).

SOY UNA CAJA

Media Queries: Son reglas para que la página cambie si la ves en un celular o en una tele grande. "Si la pantalla es chica, haz la letra grande".

Ref: CSS-Tricks Box Model & Media Queries

Layout: Flex y Grid

Flex: Una fila de gente alineada fácil.

Grid: Un tablero de ajedrez con filas y columnas.

Ref: CSS Layout Guide
